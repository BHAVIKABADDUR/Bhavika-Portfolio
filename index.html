Frame 03

What I've built

Six projects, roughly in order of how ambitious they got. Not all of them are AI — I think that's a feature, not a gap.

01 flagship

FinSight Enterprise AI

Multi-agent financial intelligence platform

My biggest build so far. It reads financial documents — bank statements, invoices — through OCR and LLM extraction, then routes anything unusual to a 5-agent LangGraph pipeline for review. It processes 387+ transactions across multiple business entities, has its own governance layer (4 custom MCP tool servers so agents never touch the database directly), and a Power BI layer so someone non-technical can just ask questions about the data.

Built with: LangGraph · MCP · n8n · Supabase · Qdrant · Groq (Llama 3.3) · Power BI · PySpark · Tesseract OCR
